Podcast Narutando no Código criado utilizando as IAs Chatgpt e Elevenlabs

Chatgpt e Elevenlabs

## 🧐 Processo de Criação
O roteiro, a capa, titulo e sub-titulo foram criados utilizando o chatgpt.

O Podcast é um exemplo interessante e atrativo de como as IAs podem revolucionar o modo como se produz contéudo digitais e apresentam uma capacidade exponencial de atrair cada vez mais públicos para estes produtos.
